# HTML5 Banner Automation

## Overview
HTML5 Banner Automation is a React and Node.js based application that simplifies the creation and editing of HTML5 banner ads. Users can modify text, replace images, preview changes in real time, and export production-ready banner packages.

## Features
- Dynamic text editing
- Image and logo replacement
- Live HTML5 banner preview
- Editable banner elements (e.g., orange strip)
- Multiple banner size support
- Export HTML5 banner as ZIP
- Real-time preview synchronization

## Tech Stack
- React
- Node.js
- Express.js
- JavaScript
- HTML5
- CSS
- GSAP
